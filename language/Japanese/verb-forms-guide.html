日语动词形式与用法详解 ​

一、先判断动词属于哪一类 ​

日语动词变形的第一步，不是背某一个词怎么变，而是先判断它属于哪一类。类别判断对了，后面的 ます形、ない形、て形、た形、可能形、被动形、使役形等才有规律可循。

日语动词通常分为三类：

  • 一类动词：五段动词。
  • 二类动词：一段动词。
  • 三类动词：变格动词，只有 する 和 来る（くる）。

1. 一类动词：五段动词 ​

一类动词的词尾必须是 う段 假名：

text
う・く・す・つ・ぬ・む・る・ぐ・ぶ

判断规则：

  • 词尾不是 る 的动词，基本都可以判断为一类动词。
  • 词尾是 る 时，要看倒数第二个假名。
  • 如果倒数第二个假名是 あ段、う段、お段，就是一类动词。

例子：

动词原形读音判断类别
書く（かく）かく词尾 く，不是 る一类
話す（はなす）はなす词尾 す，不是 る一类
飲む（のむ）のむ词尾 む，不是 る一类
登る（のぼる）のぼるる 前是 ぼ，属于 お段一类
作る（つくる）つくるる 前是 く，属于 う段一类

2. 二类动词：一段动词 ​

二类动词有两个典型特征：

  • 词尾必须是 る。
  • る 前面的假名通常是 い段 或 え段。

例子：

动词原形读音判断类别
食べる（たべる）たべるる 前是 べ，属于 え段二类
見る（みる）みるる 前是 み，属于 い段二类
起きる（おきる）おきるる 前是 き，属于 い段二类
教える（おしえる）おしえるる 前是 え，属于 え段二类

不过，る 前是 い段 或 え段 并不等于一定是二类。有些常见动词长得像二类，但实际上是一类，需要单独记忆。

常见例外：

text
帰る（かえる）・切る（きる）・走る（はしる）・知る（しる）
要る（いる）・入る（はいる）・減る（へる）・喋る（しゃべる）

记忆口诀：

text
归切走知，要入减少。

也就是：

  • 帰る（かえる）：回去。
  • 切る（きる）：切。
  • 走る（はしる）：跑。
  • 知る（しる）：知道。
  • 要る（いる）：需要。
  • 入る（はいる）：进入。
  • 減る（へる）：减少。
  • 喋る（しゃべる）：说话。

这些词虽然看起来像二类，但变形时按一类动词处理。

3. 三类动词：变格动词 ​

三类动词只有两个：

text
する：做
来る（くる）：来

它们不完全遵守一类、二类的规则，需要单独记忆。不过数量很少，重点是把常用形式背熟。

二、动词变形的核心思路 ​

日语动词变形可以理解为“词尾替换”。

一类动词最重要的是五段变化：

text
あ段：ない形、被动形、使役形
い段：ます形
う段：原形
え段：假定形、命令形、可能形
お段：意志形

以 書く（かく） 为例：

段变化例子
あ段く → か書かない（かかない）、書かれる（かかれる）、書かせる（かかせる）
い段く → き書きます（かきます）
う段原形書く（かく）
え段く → け書けば（かけば）、書け（かけ）、書ける（かける）
お段く → こ書こう（かこう）

二类动词的思路更简单：

text
去掉词尾「る」+ 后缀

以 食べる（たべる） 为例：

text
食べる（たべる） → 食べ（たべ） + ます = 食べます（たべます）
食べる（たべる） → 食べ（たべ） + ない = 食べない（たべない）
食べる（たべる） → 食べ（たべ） + た = 食べた（たべた）

三类动词主要靠记忆：

text
する → します・しない・した・して
来る（くる） → きます・こない・きた・きて

三、动词核心变形总表 ​

以下以 書く（かく）、食べる（たべる）、する、来る（くる） 为例。

形式基本意义一类：書く（かく）二类：食べる（たべる）する来る（くる）
原形非过去、辞书形書く（かく）食べる（たべる）する来る（くる）
ます形礼貌表达書きます（かきます）食べます（たべます）しますきます
ない形否定書かない（かかない）食べない（たべない）しないこない
た形过去、完成書いた（かいた）食べた（たべた）したきた
て形连接、请求、进行書いて（かいて）食べて（たべて）してきて
意志形打算、邀请書こう（かこう）食べよう（たべよう）しようこよう
假定形如果書けば（かけば）食べれば（たべれば）すればくれば
命令形命令書け（かけ）食べろ（たべろ）しろこい
可能形能、会書ける（かける）食べられる（たべられる）できるこられる
被动形被、受到書かれる（かかれる）食べられる（たべられる）されるこられる
使役形让、使書かせる（かかせる）食べさせる（たべさせる）させるこさせる
使役被动形被迫做書かせられる（かかせられる）食べさせられる（たべさせられる）させられるこさせられる

注意：上表实际列出了 11 种常用变形，不含原形则是 11 种变形形式。学习时不必纠结数量，重点是掌握每种形式的功能和构造。

四、各种动词形式的构造与用法 ​

1. 原形：辞书形、普通形 ​

原形就是字典里看到的形式，也叫辞书形。

用途：

  • 表示现在或将来的动作。
  • 用在普通体句子中。
  • 可以直接修饰名词。
  • 可以接很多语法结构，如 こと、つもり、前に。

例句：

text
毎日、日本語を勉強する。（まいにち、にほんごをべんきょうする。）
每天学习日语。
text
明日、友達に会う。（あした、ともだちにあう。）
明天见朋友。
text
本を読む時間がない。（ほんをよむじかんがない。）
没有读书的时间。

这里 読む（よむ） 直接修饰 時間（じかん），意思是“读书的时间”。

常见搭配：

结构意思例句
动词原形 + こと做某事日本語を話すことが好きです。（にほんごをはなすことがすきです。）
动词原形 + 前に在做某事之前寝る前に、歯を磨きます。（ねるまえに、はをみがきます。）
动词原形 + つもり打算做来年、日本へ行くつもりです。（らいねん、にほんへいくつもりです。）
动词原形 + ことができる能做漢字を読むことができます。（かんじをよむことができます。）

2. ます形：礼貌表达 ​

ます形 是日语中最常用的礼貌形式，常用于对不熟的人、老师、上司、客户，或者正式场合。

构造：

类别规则例子
一类词尾 う段 → い段 + ます書く（かく） → 書きます（かきます）
二类去掉 る + ます食べる（たべる） → 食べます（たべます）
三类特殊变化する → します、来る（くる） → きます

例句：

text
私は毎朝コーヒーを飲みます。（わたしはまいあさコーヒーをのみます。）
我每天早上喝咖啡。
text
昼ご飯を食べます。（ひるごはんをたべます。）
吃午饭。
text
明日、宿題をします。（あした、しゅくだいをします。）
明天做作业。

ます形 还可以继续变化：

形式意思例子
ます非过去肯定書きます（かきます）
ません非过去否定書きません（かきません）
ました过去肯定書きました（かきました）
ませんでした过去否定書きませんでした（かきませんでした）

例句：

text
昨日、映画を見ました。（きのう、えいがをみました。）
昨天看了电影。
text
今朝、朝ご飯を食べませんでした。（けさ、あさごはんをたべませんでした。）
今天早上没吃早饭。

3. ない形：否定、不做 ​

ない形 表示“不做某事”，属于普通体否定。

构造：

类别规则例子
一类词尾 う段 → あ段 + ない書く（かく） → 書かない（かかない）
一类特殊う → わ + ない買う（かう） → 買わない（かわない）
二类去掉 る + ない食べる（たべる） → 食べない（たべない）
三类特殊变化する → しない、来る（くる） → こない

例句：

text
今日は学校へ行かない。（きょうはがっこうへいかない。）
今天不去学校。
text
肉を食べない人もいます。（にくをたべないひともいます。）
也有人不吃肉。
text
彼は宿題をしない。（かれはしゅくだいをしない。）
他不做作业。

常见搭配：

结构意思例句
ないでください请不要ここで写真を撮らないでください。（ここでしゃしんをとらないでください。）
なければならない必须勉強しなければなりません。（べんきょうしなければなりません。）
なくてもいい不做也可以明日来なくてもいいです。（あしたこなくてもいいです。）
ないほうがいい最好不要夜遅く食べないほうがいいです。（よるおそくたべないほうがいいです。）

4. た形：过去、完成、经验 ​

た形 表示动作已经发生或完成，是普通体过去肯定形式。

构造：

类别规则例子
一类按音便规则变化書く（かく） → 書いた（かいた）
二类去掉 る + た食べる（たべる） → 食べた（たべた）
三类特殊变化する → した、来る（くる） → きた

例句：

text
昨日、友達に会った。（きのう、ともだちにあった。）
昨天见了朋友。
text
朝ご飯を食べた。（あさごはんをたべた。）
吃了早饭。
text
週末に掃除をした。（しゅうまつにそうじをした。）
周末打扫了卫生。

常见搭配：

结构意思例句
たことがある曾经做过日本へ行ったことがあります。（にほんへいったことがあります。）
たほうがいい最好做早く寝たほうがいいです。（はやくねたほうがいいです。）
た後で做完之后ご飯を食べた後で、散歩します。（ごはんをたべたあとで、さんぽします。）
たり、たりする列举动作週末は映画を見たり、本を読んだりします。（しゅうまつはえいがをみたり、ほんをよんだりします。）

5. て形：连接、请求、进行状态 ​

て形 是日语中最重要的连接形式之一。它本身不表示时态，而是把动词连接到后面的表达。

构造：

类别规则例子
一类按音便规则变化書く（かく） → 書いて（かいて）
二类去掉 る + て食べる（たべる） → 食べて（たべて）
三类特殊变化する → して、来る（くる） → きて

主要用法：

结构意思例句
てください请做ここに名前を書いてください。（ここになまえをかいてください。）
ている正在做、状态持续今、本を読んでいます。（いま、ほんをよんでいます。）
てもいい可以做ここに座ってもいいです。（ここにすわってもいいです。）
てはいけない不可以做ここでタバコを吸ってはいけません。（ここでタバコをすってはいけません。）
てから做完之后再手を洗ってから、ご飯を食べます。（てをあらってから、ごはんをたべます。）
てみる试着做この料理を食べてみます。（このりょうりをたべてみます。）
てしまう完成、遗憾財布を忘れてしまいました。（さいふをわすれてしまいました。）

例句：

text
窓を開けてください。（まどをあけてください。）
请打开窗户。
text
彼は今、電話をしています。（かれはいま、でんわをしています。）
他现在正在打电话。
text
宿題をしてから、ゲームをします。（しゅくだいをしてから、ゲームをします。）
做完作业之后再玩游戏。

6. 意志形：打算、邀请、提议 ​

意志形表示说话人的意愿，也可以用于邀请或提议。

构造：

类别规则例子
一类词尾 う段 → お段 + う書く（かく） → 書こう（かこう）
二类去掉 る + よう食べる（たべる） → 食べよう（たべよう）
三类特殊变化する → しよう、来る（くる） → こよう

例句：

text
今日は早く寝よう。（きょうははやくねよう。）
今天早点睡吧。
text
一緒に昼ご飯を食べよう。（いっしょにひるごはんをたべよう。）
一起吃午饭吧。
text
週末に掃除をしようと思います。（しゅうまつにそうじをしようとおもいます。）
我打算周末打扫卫生。

常见搭配：

结构意思例句
意志形 + と思う打算做日本語を勉強しようと思います。（にほんごをべんきょうしようとおもいます。）
意志形 + とする正要做出かけようとした時、電話が来ました。（でかけようとしたとき、でんわがきました。）

7. 假定形：如果 ​

假定形表示“如果做某事，就……”，常见形式是 ば形。

构造：

类别规则例子
一类词尾 う段 → え段 + ば書く（かく） → 書けば（かけば）
二类去掉 る + れば食べる（たべる） → 食べれば（たべれば）
三类特殊变化する → すれば、来る（くる） → くれば

例句：

text
時間があれば、映画を見ます。（じかんがあれば、えいがをみます。）
如果有时间，就看电影。
text
この薬を飲めば、よくなります。（このくすりをのめば、よくなります。）
如果吃这个药，就会好起来。
text
分からなければ、先生に聞いてください。（わからなければ、せんせいにきいてください。）
如果不明白，请问老师。

补充：

  • ば 更强调条件本身。
  • 如果后句是命令、请求、意志表达，使用时要注意语气和限制。

8. 命令形：强制命令 ​

命令形语气很强，常用于上级对下级、口号、紧急情况、男性化表达、固定标语等。日常会话中如果直接使用，容易显得粗鲁。

构造：

类别规则例子
一类词尾 う段 → え段書く（かく） → 書け（かけ）
二类去掉 る + ろ，也可用 よ食べる（たべる） → 食べろ（たべろ）
三类特殊变化する → しろ、来る（くる） → こい

例句：

text
早く行け。（はやくいけ。）
快去。
text
ちゃんと食べろ。（ちゃんとたべろ。）
好好吃。
text
静かにしろ。（しずかにしろ。）
安静。

更礼貌的替代表达：

text
早く行ってください。（はやくいってください。）
请快点去。
text
静かにしてください。（しずかにしてください。）
请安静。

9. 可能形：能做、会做 ​

可能形表示能力或可能性，相当于“能……”“会……”。

构造：

类别规则例子
一类词尾 う段 → え段 + る書く（かく） → 書ける（かける）
二类去掉 る + られる食べる（たべる） → 食べられる（たべられる）
三类特殊变化する → できる、来る（くる） → こられる

例句：

text
私は日本語が少し話せます。（わたしはにほんごがすこしはなせます。）
我会说一点日语。
text
この店では新鮮な魚が食べられます。（このみせではしんせんなさかながたべられます。）
在这家店能吃到新鲜的鱼。
text
明日、来られますか。（あした、こられますか。）
明天能来吗？

注意：

  • 可能形句子里，动作对象常用 が，也可以用 を。
  • 初学阶段记住 日本語を話す（にほんごをはなす） 变成 日本語が話せる（にほんごがはなせる） 很有帮助。

对比：

text
日本語を話します。（にほんごをはなします。）
说日语。
text
日本語が話せます。（にほんごがはなせます。）
会说日语。

10. 被动形：被做、受到影响 ​

被动形表示主语受到某个动作的影响。日语被动不仅表示“被……”，还常表示“受害、困扰、受到影响”。

构造：

类别规则例子
一类词尾 う段 → あ段 + れる書く（かく） → 書かれる（かかれる）
一类特殊う → わ + れる買う（かう） → 買われる（かわれる）
二类去掉 る + られる食べる（たべる） → 食べられる（たべられる）
三类特殊变化する → される、来る（くる） → こられる

例句：

text
先生に名前を呼ばれました。（せんせいになまえをよばれました。）
被老师叫了名字。
text
私は友達に笑われました。（わたしはともだちにわらわれました。）
我被朋友笑了。
text
雨に降られて、服が濡れました。（あめにふられて、ふくがぬれました。）
被雨淋了，衣服湿了。

第三句是日语常见的“受害被动”：并不是雨有意淋我，而是说话人受到了这个事件的影响。

常见句型：

text
A は B に 动词被动形
A 被 B 做了某事

例子：

text
私は母に叱られました。（わたしはははにしかられました。）
我被妈妈骂了。

11. 使役形：让某人做、使某事发生 ​

使役形表示“让某人做某事”“使某人做某事”。

构造：

类别规则例子
一类词尾 う段 → あ段 + せる書く（かく） → 書かせる（かかせる）
一类特殊う → わ + せる買う（かう） → 買わせる（かわせる）
二类去掉 る + させる食べる（たべる） → 食べさせる（たべさせる）
三类特殊变化する → させる、来る（くる） → こさせる

例句：

text
母は子どもに野菜を食べさせました。（はははこどもにやさいをたべさせました。）
妈妈让孩子吃蔬菜。
text
先生は学生に作文を書かせました。（せんせいはがくせいにさくぶんをかかせました。）
老师让学生写作文。
text
上司は私に残業させました。（じょうしはわたしにざんぎょうさせました。）
上司让我加班。

常见结构：

结构用法例句
人に + 自动词使役让某人做某动作子どもに走らせる。（こどもにはしらせる。）
人に + 他动词宾语を + 使役让某人做某事学生に本を読ませる。（がくせいにほんをよませる。）

12. 使役被动形：被迫做 ​

使役被动形表示“被迫做某事”“不得不做某事”，语气通常带有不情愿。

构造：

类别规则例子
一类词尾 う段 → あ段 + せられる書く（かく） → 書かせられる（かかせられる）
二类去掉 る + させられる食べる（たべる） → 食べさせられる（たべさせられる）
三类特殊变化する → させられる、来る（くる） → こさせられる

例句：

text
子どものころ、母にピアノを習わせられました。（こどものころ、ははにピアノをならわせられました。）
小时候，我被妈妈逼着学钢琴。
text
会社で長いレポートを書かせられました。（かいしゃでながいレポートをかかせられました。）
在公司被迫写了很长的报告。
text
毎日、残業させられています。（まいにち、ざんぎょうさせられています。）
每天都被迫加班。

一类动词的使役被动在口语中常有缩约形式：

原形标准形式口语缩约
書く（かく）書かせられる（かかせられる）書かされる（かかされる）
飲む（のむ）飲ませられる（のませられる）飲まされる（のまされる）
待つ（まつ）待たせられる（またせられる）待たされる（またされる）

例句：

text
先生に作文を書かされました。（せんせいにさくぶんをかかされました。）
被老师逼着写作文。

五、一类动词 て形、た形 音便规则 ​

一类动词的 て形 和 た形 是变形中最容易出错的部分。关键是按词尾分类记忆。

1. く、ぐ ​

词尾て形た形例子
くいていた書く（かく） → 書いて（かいて） → 書いた（かいた）
ぐいでいだ泳ぐ（およぐ） → 泳いで（およいで） → 泳いだ（およいだ）

例句：

text
ノートに名前を書いてください。（ノートになまえをかいてください。）
请在笔记本上写名字。
text
昨日、プールで泳いだ。（きのう、プールでおよいだ。）
昨天在泳池游泳了。

2. う、つ、る ​

词尾て形た形例子
うってった買う（かう） → 買って（かって） → 買った（かった）
つってった待つ（まつ） → 待って（まって） → 待った（まった）
るってった帰る（かえる） → 帰って（かえって） → 帰った（かえった）

例句：

text
駅で友達を待っている。（えきでともだちをまっている。）
正在车站等朋友。
text
昨日、家へ帰った。（きのう、いえへかえった。）
昨天回家了。

3. む、ぶ、ぬ ​

词尾て形た形例子
むんでんだ読む（よむ） → 読んで（よんで） → 読んだ（よんだ）
ぶんでんだ遊ぶ（あそぶ） → 遊んで（あそんで） → 遊んだ（あそんだ）
ぬんでんだ死ぬ（しぬ） → 死んで（しんで） → 死んだ（しんだ）

例句：

text
毎晩、本を読んでいます。（まいばん、ほんをよんでいます。）
每天晚上读书。
text
子どもたちは公園で遊んだ。（こどもたちはこうえんであそんだ。）
孩子们在公园玩了。

4. す ​

词尾て形た形例子
すしてした話す（はなす） → 話して（はなして） → 話した（はなした）

例句：

text
先生と日本語で話してみました。（せんせいとにほんごではなしてみました。）
试着和老师用日语说了话。

5. 行く（いく） 的特殊音便 ​

行く（いく） 是最重要的特殊例外。

text
行く（いく） → 行って（いって） → 行った（いった）

它不按 く → いて / いた 的规则变化。

例句：

text
昨日、図書館へ行った。（きのう、としょかんへいった。）
昨天去了图书馆。
text
一緒に行ってください。（いっしょにいってください。）
请一起去。

六、常见形式之间的区别 ​

1. 食べられる（たべられる） 为什么既可能是可能形，也可能是被动形？ ​

二类动词的可能形和被动形形式相同。

text
食べる（たべる） → 食べられる（たべられる）

它可能表示：

  • 能吃。
  • 被吃。

需要根据句子判断。

text
この魚は食べられます。（このさかなはたべられます。）
这条鱼可以吃。

这里更自然地理解为“可以吃”。

text
魚が猫に食べられました。（さかながねこにたべられました。）
鱼被猫吃了。

这里有 猫に（ねこに），表示动作发出者，所以是被动。

2. ている 不只表示正在进行 ​

ている 有两大常见用法：

  • 动作正在进行。
  • 动作结果或状态持续。

正在进行：

text
今、雨が降っています。（いま、あめがふっています。）
现在正在下雨。

状态持续：

text
窓が開いています。（まどがあいています。）
窗户开着。
text
彼は結婚しています。（かれはけっこんしています。）
他已经结婚了。

結婚しています（けっこんしています） 不是“正在结婚”，而是“处于已婚状态”。

3. た形 不一定只表示过去 ​

た形 可以表示过去，也可以出现在固定语法中。

过去：

text
昨日、映画を見た。（きのう、えいがをみた。）
昨天看了电影。

经验：

text
日本へ行ったことがあります。（にほんへいったことがあります。）
去过日本。

建议：

text
もっと練習したほうがいいです。（もっとれんしゅうしたほうがいいです。）
最好多练习。

动作完成后：

text
宿題をした後で、テレビを見ます。（しゅくだいをしたあとで、テレビをみます。）
做完作业后看电视。

4. ない形 和 ません 的区别 ​

两者都能表示否定，但语体不同。

形式语体例子
ない形普通体食べない（たべない）
ません礼貌体食べません（たべません）

对比：

text
今日は肉を食べない。（きょうはにくをたべない。）
今天不吃肉。
text
今日は肉を食べません。（きょうはにくをたべません。）
今天不吃肉。

意思接近，但后者更礼貌。

七、特殊例外速查 ​

1. する ​

形式变化
原形する
ます形します
ない形しない
た形した
て形して
意志形しよう
假定形すれば
命令形しろ / せよ
可能形できる
被动形される
使役形させる
使役被动形させられる

例句：

text
毎日、勉強します。（まいにち、べんきょうします。）
每天学习。
text
この仕事は一人でできます。（このしごとはひとりでできます。）
这个工作一个人能做。

2. 来る（くる） ​

形式变化
原形来る（くる）
ます形来ます（きます）
ない形来ない（こない）
た形来た（きた）
て形来て（きて）
意志形来よう（こよう）
假定形来れば（くれば）
命令形来い（こい）
可能形来られる（こられる）
被动形来られる（こられる）
使役形来させる（こさせる）
使役被动形来させられる（こさせられる）

例句：

text
明日、友達が家に来ます。（あした、ともだちがいえにきます。）
明天朋友来我家。
text
明日、来られますか。（あした、こられますか。）
明天能来吗？

3. 行く（いく） ​

行く（いく） 本身是一类动词，但 て形 和 た形 特殊。

形式变化
原形行く（いく）
ます形行きます（いきます）
ない形行かない（いかない）
て形行って（いって）
た形行った（いった）

例句：

text
週末、京都へ行きます。（しゅうまつ、きょうとへいきます。）
周末去京都。
text
昨日、京都へ行った。（きのう、きょうとへいった。）
昨天去了京都。

八、学习时最容易混淆的点 ​

1. 先判断类别，再套规则 ​

不要看到 る 就直接判断为二类。

text
見る（みる）：二类 → 見ます（みます）、見ない（みない）、見た（みた）
帰る（かえる）：一类 → 帰ります（かえります）、帰らない（かえらない）、帰った（かえった）

对比：

原形类别ます形ない形た形
見る（みる）二类見ます（みます）見ない（みない）見た（みた）
帰る（かえる）一类帰ります（かえります）帰らない（かえらない）帰った（かえった）

2. 一类动词 う 变 ない形 时是 わない ​

text
買う（かう） → 買わない（かわない）
会う（あう） → 会わない（あわない）
言う（いう） → 言わない（いわない）

不能写成：

text
買あない（かあない）
会あない（ああない）
言あない（いあない）

3. 二类动词的 られる 可能有多个含义 ​

text
食べられる（たべられる）
見られる（みられる）
起きられる（おきられる）

可能表示“能做”，也可能表示“被做”。判断时看上下文，尤其看是否出现动作发出者 に。

4. 命令形语气很强 ​

日常请求优先使用：

text
てください
てもらえますか
ていただけますか

比如：

text
座れ。（すわれ。）
坐下。

这句话很强硬。

更自然的表达：

text
座ってください。（すわってください。）
请坐。
text
座っていただけますか。（すわっていただけますか。）
可以请您坐下吗？

九、速查口诀 ​

text
一类看尾巴，あいうえお换；
二类去る就行，三类就俩王。

ます用い段，ない用あ段；
ば形和命令，多数走え段；
意志变お段，再加一个う。

て形音便记：
つるう变って，
むぶぬ变んで，
くぐ变いていで，
す就变して。

行く（いく）是例外，
行って（いって）、行った（いった）别忘记。

十一、最终总结 ​

日语动词变形可以按三个步骤学习：

text
第一步：判断动词类别。
第二步：按类别套变形规则。
第三步：把变形放进句型里理解用法。

最重要的不是一次背完所有形式，而是先掌握以下基础链条：

text
原形 → ます形 → ない形 → て形 → た形

之后再逐步扩展：

text
意志形 → 假定形 → 可能形 → 被动形 → 使役形 → 使役被动形

只要能稳定判断一类、二类、三类，动词变形就会从“死记硬背”变成“按规则替换词尾”。
