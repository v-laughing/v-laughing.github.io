日语高频 100 词用法与例句 ​

使用说明 ​

这份文档整理了日语中非常高频的 100 个词。它们包括助词、助动词、基础动词、代词、时间词、常见名词和形容词。

学习时建议不要只背中文意思，而要重点看它们在句子中的位置和作用。尤其是 は、が、を、に、で 这类助词，本身不表示具体事物，却决定了句子的结构。

一、助词与助动词 ​

1. の（读音：の） ​

核心用法：连接名词，表示所属、性质、内容，也可以把前面的内容名词化。

text
これは私の本です。
这是我的书。

私の本 表示“我的书”，の 把 私 和 本 连接起来。

text
日本語を勉強するのは楽しいです。
学习日语很开心。

这里的 の 把 日本語を勉強する 名词化，变成“学习日语这件事”。

2. に（读音：に） ​

核心用法：表示时间点、存在地点、动作到达点、对象或目的。

text
七時に起きます。
七点起床。

に 标记明确的时间点。

text
友達にプレゼントをあげました。
我给了朋友礼物。

友達に 表示动作到达的对象。

3. は（读音：わ，作助词时） ​

核心用法：提示话题，意思是“关于……”，也常带有对比。

text
私は学生です。
我是学生。

私は 表示“关于我，接下来说明我是学生”。

text
コーヒーは飲みますが、お茶は飲みません。
咖啡我喝，但茶我不喝。

两个 は 都带有对比感。

4. を（读音：お） ​

核心用法：标记动作直接作用的对象，也可以表示经过的场所或离开的地点。

text
本を読みます。
读书。

本を 是 読みます 的对象。

text
公園を散歩します。
在公园散步。

这里的 を 表示经过或活动的范围。

5. が（读音：が） ​

核心用法：标记主语、焦点，强调“是谁/是什么”；也可表示轻微转折。

text
雨が降っています。
正在下雨。

雨が 是自然现象的主体。

text
日本語は難しいですが、面白いです。
日语很难，但是很有趣。

这里的 が 表示转折。

6. で（读音：で） ​

核心用法：表示动作发生地点、手段、材料、范围或原因。

text
図書館で勉強します。
在图书馆学习。

で 标记动作发生的场所。

text
電車で会社へ行きます。
坐电车去公司。

で 标记交通手段。

7. と（读音：と） ​

核心用法：表示“和”、引用内容、共同动作对象。

text
友達と映画を見ました。
和朋友看了电影。

友達と 表示一起做动作的人。

text
先生は「静かにしてください」と言いました。
老师说：“请安静。”

と 标记引用内容。

8. も（读音：も） ​

核心用法：表示“也”，把前面的成分加入同一类情况。

text
私も行きます。
我也去。

も 表示“我”也属于“去”的人。

text
今日も暑いです。
今天也很热。

表示今天和之前一样热。

9. から（读音：から） ​

核心用法：表示起点、来源，也表示原因。

text
九時から授業があります。
九点开始有课。

から 表示时间起点。

text
雨ですから、出かけません。
因为下雨，所以不出门。

から 表示原因，语气较直接。

10. まで（读音：まで） ​

核心用法：表示终点、范围的结束。

text
駅まで歩きます。
走到车站。

まで 表示移动的终点。

text
五時まで働きます。
工作到五点。

まで 表示时间的截止点。

11. へ（读音：え，作助词时） ​

核心用法：表示移动方向，读作 え。

text
日本へ行きます。
去日本。

へ 强调动作的方向。

text
こちらへどうぞ。
请往这边来。

这里的 へ 表示朝向“这边”。

12. より（读音：より） ​

核心用法：表示比较基准，也可以表示“从”。

text
東京は大阪より大きいです。
东京比大阪大。

大阪より 是比较基准。

text
心より感謝します。
由衷感谢。

这里的 より 表示来源，常见于正式表达。

13. だけ（读音：だけ） ​

核心用法：表示限定，“只、仅仅”。

text
水だけ飲みました。
只喝了水。

だけ 限定对象是水，没有其他东西。

text
見るだけです。
只是看看。

表示动作限于“看”。

14. など（读音：など） ​

核心用法：表示列举，“等等、之类”。

text
机の上に本やノートなどがあります。
桌子上有书、笔记本等。

など 表示列举没有说完。

text
週末は掃除などをします。
周末会做打扫之类的事。

表示举出一个代表性例子。

15. て（读音：て） ​

核心用法：连接动作、表示方式、原因，或构成多种语法形式。

text
朝ご飯を食べて、学校へ行きます。
吃早饭后去学校。

て 连接前后两个动作。

text
この漢字を見てください。
请看这个汉字。

てください 表示请求。

16. た（读音：た） ​

核心用法：表示过去或完成。

text
昨日、映画を見ました。
昨天看了电影。

見ました 是 見る 的过去礼貌形。

text
宿題はもう終わった。
作业已经完成了。

这里的 た 表示完成。

17. だ（读音：だ） ​

核心用法：判断助动词，相当于普通体的“是”。

text
今日は休みだ。
今天是休息日。

だ 用在名词或な形容词后，语气普通。

text
彼は親切だ。
他很亲切。

親切だ 表示“是亲切的”。

18. です（读音：です） ​

核心用法：礼貌判断，接在名词、な形容词、い形容词后。

text
私は学生です。
我是学生。

です 让句子变得礼貌。

text
この店は有名です。
这家店很有名。

名词句和な形容词句常用 です 结尾。

19. ます（读音：ます） ​

核心用法：动词礼貌形结尾，用于正式或礼貌场景。

text
毎日、日本語を勉強します。
每天学习日语。

勉強します 是 勉強する 的礼貌形。

text
明日、友達に会います。
明天见朋友。

会います 比 会う 更礼貌。

20. ない（读音：ない） ​

核心用法：表示否定，“不、没有”。

text
肉を食べない。
不吃肉。

食べない 是否定形。

text
今日は時間がない。
今天没有时间。

这里的 ない 表示“不存在”。

二、基础动词 ​

21. する（读音：する） ​

核心用法：做；也常接在汉语名词后构成动词。

text
宿題をします。
做作业。

する 表示执行某个动作。

text
日本語を勉強します。
学习日语。

勉強する 是“学习”。

22. いる（读音：いる） ​

核心用法：表示人或动物的存在，也用于 ている 表示正在进行或状态持续。

text
教室に学生がいます。
教室里有学生。

いる 用于有生命的人或动物。

text
今、本を読んでいます。
现在正在读书。

読んでいます 表示动作正在进行。

23. ある（读音：ある） ​

核心用法：表示无生命事物的存在。

text
机の上に本があります。
桌子上有书。

ある 用于物品、事情等无生命对象。

text
明日、会議があります。
明天有会议。

事件和安排也常用 ある。

24. なる（读音：なる） ​

核心用法：变成、成为，表示状态变化。

text
日本語が上手になりました。
日语变好了。

上手になる 表示“变得擅长”。

text
寒くなりました。
变冷了。

い形容词接 なる 时变为 寒くなる。

25. できる（读音：できる） ​

核心用法：能够，会；也表示完成、产生。

text
私は日本語が少しできます。
我会一点日语。

能力对象常用 が 标记。

text
晩ご飯ができました。
晚饭做好了。

这里的 できる 表示完成。

26. いう（读音：いう） ​

核心用法：说；也用于“叫作、称为”。

text
彼は「ありがとう」と言いました。
他说了“谢谢”。

引用内容后接 と言う。

text
これは日本語で何と言いますか。
这个用日语怎么说？

何と言いますか 是询问表达方式的常用句。

27. 行く（读音：いく） ​

核心用法：去，表示离开当前位置朝某处移动。

text
明日、学校へ行きます。
明天去学校。

目的地可用 へ 或 に 标记。

text
買い物に行きます。
去买东西。

动词ます形去掉 ます 加 に行く 表示“去做某事”。

28. 来る（读音：くる） ​

核心用法：来，表示朝说话者或当前场景移动。

text
友達が家に来ました。
朋友来我家了。

来る 的方向通常是靠近说话者。

text
また来てください。
请再来。

来てください 是请求对方来的礼貌说法。

29. 見る（读音：みる） ​

核心用法：看、观看、查看。

text
テレビを見ます。
看电视。

見る 的对象用 を。

text
この写真を見てください。
请看这张照片。

見てください 常用于请对方查看。

30. 思う（读音：おもう） ​

核心用法：想、认为，常和 と 搭配。

text
明日は雨だと思います。
我觉得明天会下雨。

想法内容后接 と思います。

text
日本語は面白いと思います。
我觉得日语很有趣。

と思います 比直接断言更委婉。

31. 分かる（读音：わかる） ​

核心用法：明白、理解。理解的对象常用 が。

text
この問題が分かります。
我明白这个问题。

日语里常说“问题明白”，所以用 が。

text
意味が分かりません。
不明白意思。

这是学习中非常常用的一句。

32. 知る（读音：しる） ​

核心用法：知道。常用 知っている 表示“知道”这个状态。

text
その人を知っています。
我认识那个人。

知っています 表示已经知道并保持这个状态。

text
答えを知りません。
我不知道答案。

否定时常用 知りません，而不是 知っていません。

33. 食べる（读音：たべる） ​

核心用法：吃。

text
朝ご飯を食べます。
吃早饭。

吃的对象用 を 标记。

text
何を食べたいですか。
你想吃什么？

食べたい 表示“想吃”。

34. 飲む（读音：のむ） ​

核心用法：喝；也可用于吃药。

text
水を飲みます。
喝水。

飲む 的对象用 を。

text
薬を飲みました。
吃了药。

日语里“吃药”通常说 薬を飲む。

35. 使う（读音：つかう） ​

核心用法：使用。

text
このアプリを使います。
使用这个应用。

工具、软件、方法都可以用 使う。

text
日本語を使って話します。
用日语说话。

使って 表示手段或方式。

36. 入る（读音：はいる） ​

核心用法：进入；也表示加入组织、容器中有东西。

text
部屋に入ります。
进入房间。

进入的地点用 に。

text
コーヒーに砂糖が入っています。
咖啡里加了糖。

入っている 表示已经在里面的状态。

37. 出る（读音：でる） ​

核心用法：出去、出来、出现、参加。

text
家を出ます。
离开家。

离开的地点常用 を。

text
会議に出ます。
参加会议。

会議に出る 是“出席会议”。

38. 持つ（读音：もつ） ​

核心用法：拿、持有、拥有。

text
かばんを持っています。
拿着包。

持っている 表示持有或拿着的状态。

text
お金を持っていません。
没有带钱。

日常会话中常用于“有没有带/拥有”。

39. 作る（读音：つくる） ​

核心用法：制作、做、创造。

text
母は料理を作ります。
妈妈做饭。

制作出来的对象用 を。

text
新しいアカウントを作りました。
创建了新账号。

抽象对象也可以用 作る。

40. 書く（读音：かく） ​

核心用法：写、画。

text
ノートに名前を書きます。
在笔记本上写名字。

写入的位置用 に。

text
メールを書いています。
正在写邮件。

書いている 表示动作正在进行。

41. 聞く（读音：きく） ​

核心用法：听；询问。

text
音楽を聞きます。
听音乐。

听的对象用 を。

text
先生に聞いてください。
请问老师。

询问对象用 に。

42. 話す（读音：はなす） ​

核心用法：说话、交谈。

text
友達と日本語で話します。
和朋友用日语说话。

友達と 表示交谈对象，日本語で 表示使用的语言。

text
ゆっくり話してください。
请慢慢说。

話してください 是请求对方说话。

43. 読む（读音：よむ） ​

核心用法：读、阅读。

text
新聞を読みます。
读报纸。

阅读对象用 を。

text
この漢字は何と読みますか。
这个汉字怎么读？

何と読む 用于询问读法。

44. 会う（读音：あう） ​

核心用法：见面，见面的对象常用 に。

text
明日、友達に会います。
明天见朋友。

日语中通常说 人に会う。

text
駅で先生に会いました。
在车站遇到了老师。

地点用 で，见面对象用 に。

45. 買う（读音：かう） ​

核心用法：买。

text
コンビニでパンを買いました。
在便利店买了面包。

购买地点用 で，购买对象用 を。

text
新しいスマホを買いたいです。
想买新手机。

買いたい 表示“想买”。

46. 取る（读音：とる） ​

核心用法：拿、取得、拍摄、预约等。

text
机の上のペンを取ってください。
请拿桌上的笔。

取る 表示把东西拿到手。

text
写真を撮ります。
拍照。

“拍照”常写作 撮る，读音同为 とる。

47. 入れる（读音：いれる） ​

核心用法：放入、加入、打开电器。

text
かばんに本を入れます。
把书放进包里。

放入的位置用 に。

text
エアコンを入れてください。
请打开空调。

电器开关场景中，入れる 可表示“打开”。

48. 出す（读音：だす） ​

核心用法：拿出、提交、发出。

text
かばんから財布を出しました。
从包里拿出了钱包。

来源用 から，拿出的对象用 を。

text
宿題を出してください。
请提交作业。

宿題を出す 可以表示“交作业”。

49. 待つ（读音：まつ） ​

核心用法：等待。

text
駅で友達を待ちます。
在车站等朋友。

等待的对象用 を。

text
少し待ってください。
请稍等。

这是非常常用的礼貌表达。

50. 帰る（读音：かえる） ​

核心用法：回去、回家、返回原处。

text
家に帰ります。
回家。

回到的地点用 に。

text
今日は早く帰りたいです。
今天想早点回去。

帰りたい 表示“想回去”。

三、代词、疑问词与指示词 ​

51. 人（读音：ひと） ​

核心用法：人；也可以接在国家、职业等词后表示某类人。

text
あの人は先生です。
那个人是老师。

人 指具体的人。

text
日本人の友達がいます。
我有日本朋友。

日本人 表示日本人。

52. 私（读音：わたし） ​

核心用法：我，较中性、常用。

text
私は中国人です。
我是中国人。

私 可用于大多数正式和普通场合。

text
これは私のかばんです。
这是我的包。

私の 表示“我的”。

53. あなた（读音：あなた） ​

核心用法：你。日语中不常频繁使用，能省略时常省略。

text
あなたは学生ですか。
你是学生吗？

直接称呼对方时可以用，但有时会显得距离感较强。

text
あなたの名前は何ですか。
你的名字是什么？

学习阶段常见，实际会话中也常直接问 お名前は何ですか。

54. これ（读音：これ） ​

核心用法：这个，指靠近说话者的事物。

text
これは何ですか。
这是什么？

これ 指眼前靠近自己的一件东西。

text
これをください。
请给我这个。

购物时非常常用。

55. それ（读音：それ） ​

核心用法：那个，指靠近听话者，或前文提到的事物。

text
それはいくらですか。
那个多少钱？

それ 指靠近对方的东西。

text
それはいい考えですね。
那是个好主意。

这里的 それ 指前面提到的想法。

56. あれ（读音：あれ） ​

核心用法：那个，指离说话者和听话者都远的事物。

text
あれは富士山です。
那是富士山。

あれ 表示远处的东西。

text
あれを見てください。
请看那个。

用于把对方注意力引向远处。

57. ここ（读音：ここ） ​

核心用法：这里，靠近说话者的位置。

text
ここは図書館です。
这里是图书馆。

ここ 指当前所在或靠近自己的地方。

text
ここに名前を書いてください。
请在这里写名字。

位置后接 に 表示写入点。

58. そこ（读音：そこ） ​

核心用法：那里，靠近听话者的位置，或前文提到的地点。

text
そこに座ってください。
请坐在那里。

そこ 指对方附近的位置。

text
その店は有名です。そこへ行きたいです。
那家店很有名。我想去那里。

第二句的 そこ 指前文的店。

59. あそこ（读音：あそこ） ​

核心用法：那里，离双方都较远的地点。

text
あそこに駅があります。
那边有车站。

あそこ 指远处地点。

text
あそこで写真を撮りましょう。
在那里拍照吧。

で 标记动作发生地点。

60. どこ（读音：どこ） ​

核心用法：哪里。

text
トイレはどこですか。
洗手间在哪里？

询问地点时常用。

text
どこへ行きますか。
你要去哪里？

移动方向可用 へ。

61. 何（读音：なに／なん） ​

核心用法：什么。读音有 なに 和 なん 两种。

text
これは何ですか。
这是什么？

在 です 前通常读作 なん。

text
何を食べますか。
你吃什么？

在 を 前通常读作 なに。

62. 誰（读音：だれ） ​

核心用法：谁。

text
あの人は誰ですか。
那个人是谁？

询问人的身份。

text
誰と行きますか。
你和谁去？

誰と 表示“和谁”。

63. いつ（读音：いつ） ​

核心用法：什么时候。

text
誕生日はいつですか。
生日是什么时候？

询问时间。

text
いつ日本へ行きますか。
你什么时候去日本？

いつ 不需要加 に。

64. どう（读音：どう） ​

核心用法：怎样、如何。

text
日本の生活はどうですか。
日本的生活怎么样？

询问状态或评价。

text
これはどう使いますか。
这个怎么使用？

询问方法时也用 どう。

65. どれ（读音：どれ） ​

核心用法：哪个，用于从多个东西中选择。

text
あなたの傘はどれですか。
你的伞是哪一把？

どれ 单独作名词使用。

text
どれが一番好きですか。
你最喜欢哪个？

被选择的对象用 が 标记。

66. この（读音：この） ​

核心用法：这个，必须接名词。

text
この本は面白いです。
这本书很有趣。

この 后面必须接名词，不能单独使用。

text
この人は私の先生です。
这个人是我的老师。

この人 指靠近说话者或正在介绍的人。

67. その（读音：その） ​

核心用法：那个，必须接名词，指靠近听话者或前文提到的事物。

text
そのかばんは誰のですか。
那个包是谁的？

その 后接名词。

text
昨日、新しい店へ行きました。その店は安かったです。
昨天去了新店。那家店很便宜。

第二句的 その店 指前文提到的店。

68. あの（读音：あの） ​

核心用法：那个，必须接名词，指远处或双方都知道的对象。

text
あの建物は学校です。
那栋建筑是学校。

指远处的建筑。

text
あの人を知っていますか。
你认识那个人吗？

あの人 可以指双方都看得见或都知道的人。

69. どの（读音：どの） ​

核心用法：哪个，必须接名词。

text
どの本が好きですか。
你喜欢哪本书？

どの 后面一定接名词。

text
どの駅で降りますか。
在哪个车站下车？

询问具体选择。

四、时间词与常见名词 ​

70. 今日（读音：きょう） ​

核心用法：今天。

text
今日は暑いです。
今天很热。

时间词作话题时常接 は。

text
今日、友達に会います。
今天见朋友。

时间背景可以直接放在句首。

71. 明日（读音：あした／あす） ​

核心用法：明天。

text
明日は休みです。
明天休息。

明日は 把明天设为话题。

text
明日、試験があります。
明天有考试。

試験がある 表示“有考试”。

72. 昨日（读音：きのう） ​

核心用法：昨天。

text
昨日、映画を見ました。
昨天看了电影。

过去时间常和过去形搭配。

text
昨日は雨でした。
昨天是雨天。

名词句的过去礼貌形是 でした。

73. 今（读音：いま） ​

核心用法：现在。

text
今、何をしていますか。
你现在在做什么？

今 常和 ている 搭配表示正在进行。

text
今から始めます。
现在开始。

今から 表示“从现在起”。

74. 時間（读音：じかん） ​

核心用法：时间、小时。

text
今日は時間がありません。
今天没有时间。

時間がある/ない 表示有/没有时间。

text
二時間勉強しました。
学习了两个小时。

表示时长时可直接放在动词前。

75. 年（读音：とし／ねん） ​

核心用法：年；也表示年龄、年份相关概念。

text
一年に一回旅行します。
一年旅行一次。

一年に一回 表示频率。

text
今年は忙しいです。
今年很忙。

今年 是“今年”。

76. 月（读音：つき／がつ／げつ） ​

核心用法：月、月份；读音根据意思不同而变化。

text
来月、日本へ行きます。
下个月去日本。

来月 是“下个月”。

text
月がきれいです。
月亮很美。

表示月亮时读作 つき。

77. 日（读音：ひ／にち／び） ​

核心用法：日、天、日期。

text
毎日、日本語を勉強します。
每天学习日语。

毎日 是“每天”。

text
今日は何日ですか。
今天几号？

询问日期时用 何日。

78. 会社（读音：かいしゃ） ​

核心用法：公司。

text
父は会社で働いています。
父亲在公司工作。

工作地点用 で。

text
会社へ行きます。
去公司。

移动方向用 へ 或 に。

79. 学校（读音：がっこう） ​

核心用法：学校。

text
学校で日本語を勉強します。
在学校学习日语。

动作发生地点用 で。

text
学校は駅の近くにあります。
学校在车站附近。

无生命地点存在用 あります。

80. 家（读音：いえ／うち） ​

核心用法：家、房子。读作 いえ 或 うち。

text
家に帰ります。
回家。

家に帰る 是固定常用表达。

text
私の家は駅から近いです。
我家离车站近。

私の家 表示“我的家”。

81. 友達（读音：ともだち） ​

核心用法：朋友。

text
友達と昼ご飯を食べました。
和朋友吃了午饭。

一起做事的人用 と。

text
友達に電話しました。
给朋友打了电话。

动作对象用 に。

82. 先生（读音：せんせい） ​

核心用法：老师、医生、律师等受尊敬职业的称呼。

text
先生に質問しました。
问了老师一个问题。

询问对象用 に。

text
田中先生は日本語を教えています。
田中老师在教日语。

称呼别人时常把姓和 先生 连用。

83. 学生（读音：がくせい） ​

核心用法：学生，通常指大学生或广义学生。

text
私は学生です。
我是学生。

身份判断句常用 名词 + です。

text
教室に学生がいます。
教室里有学生。

有生命对象存在用 います。

84. 日本（读音：にほん／にっぽん） ​

核心用法：日本。

text
日本へ行きたいです。
想去日本。

目的地用 へ。

text
日本の文化に興味があります。
我对日本文化感兴趣。

日本の文化 表示“日本的文化”。

85. 日本語（读音：にほんご） ​

核心用法：日语。

text
日本語を勉強しています。
正在学习日语。

学习对象用 を。

text
日本語で話してください。
请用日语说。

语言作为手段时用 で。

86. 仕事（读音：しごと） ​

核心用法：工作、事情。

text
仕事が忙しいです。
工作很忙。

仕事が忙しい 是常见表达。

text
明日、仕事があります。
明天有工作。

安排或事情用 あります。

87. 勉強（读音：べんきょう） ​

核心用法：学习；加 する 构成动词。

text
毎日、勉強します。
每天学习。

勉強する 是“学习”。

text
日本語の勉強は楽しいです。
日语学习很开心。

日本語の勉強 表示“日语的学习”。

88. 電話（读音：でんわ） ​

核心用法：电话；打电话常说 電話する。

text
母に電話します。
给妈妈打电话。

打电话的对象用 に。

text
電話番号を教えてください。
请告诉我电话号码。

電話番号 是“电话号码”。

89. 名前（读音：なまえ） ​

核心用法：名字。

text
名前を書いてください。
请写名字。

写的对象用 を，这里省略了 を 也很自然。

text
お名前は何ですか。
您的名字是什么？

お名前 比 名前 更礼貌。

90. お金（读音：おかね） ​

核心用法：钱。前缀 お 让表达更自然、礼貌。

text
お金がありません。
没有钱。

お金がある/ない 表示有/没有钱。

text
お金を払います。
付钱。

支付对象用 を。

五、常见形容词 ​

91. 大きい（读音：おおきい） ​

核心用法：大的。

text
この部屋は大きいです。
这个房间很大。

い形容词可以直接接 です。

text
もっと大きいサイズはありますか。
有更大的尺码吗？

大きい + 名词 直接修饰名词。

92. 小さい（读音：ちいさい） ​

核心用法：小的。

text
このかばんは小さいです。
这个包很小。

描述物体大小。

text
小さい声で話してください。
请小声说话。

小さい声 表示“小的声音、低声”。

93. 新しい（读音：あたらしい） ​

核心用法：新的。

text
新しい本を買いました。
买了新书。

直接修饰名词。

text
このスマホは新しいです。
这部手机是新的。

作谓语时接 です。

94. 古い（读音：ふるい） ​

核心用法：旧的、古老的。

text
これは古い写真です。
这是旧照片。

古い 可以表示年代久。

text
この建物はとても古いです。
这栋建筑很古老。

用于建筑、物品、习惯等。

95. いい（读音：いい） ​

核心用法：好的、可以。否定和过去变化时通常按 よい 变化。

text
この店はいいです。
这家店不错。

いい 是日常评价中非常常见的词。

text
ここで写真を撮ってもいいですか。
可以在这里拍照吗？

てもいいですか 表示请求许可。

96. 悪い（读音：わるい） ​

核心用法：坏的、不好的；也可表示抱歉。

text
今日は天気が悪いです。
今天天气不好。

天気が悪い 是固定常用表达。

text
悪いですが、もう一度お願いします。
不好意思，请再说一遍。

会话中可作轻微道歉。

97. 多い（读音：おおい） ​

核心用法：多的。常用 名词が多い。

text
この町は人が多いです。
这个城市人很多。

日语里常说“人多”，用 人が多い。

text
今日は仕事が多いです。
今天工作很多。

表示数量多。

98. 少ない（读音：すくない） ​

核心用法：少的。常用 名词が少ない。

text
この店は客が少ないです。
这家店客人少。

客が少ない 表示客人数量少。

text
時間が少ないです。
时间很少。

常用于说明时间不够。

99. 高い（读音：たかい） ​

核心用法：高的、贵的。

text
このビルは高いです。
这栋楼很高。

表示高度高。

text
この時計は高いです。
这块表很贵。

描述价格时表示“贵”。

100. 安い（读音：やすい） ​

核心用法：便宜的；注意“安心的”是 安心，不是 安い。

text
この店は安いです。
这家店很便宜。

安い 常用于价格低。

text
もっと安いものはありますか。
有更便宜的东西吗？

购物时常用来询问更低价格的商品。

六、复习建议 ​

先重点掌握前 20 个功能词，因为它们决定句子结构。然后把 21 到 50 的基础动词配合 ます形、ない形、て形、た形 练习。最后用后 50 个代词、名词和形容词造简单句。

推荐练习方式：

text
今日、私は学校で日本語を勉強します。
今天，我在学校学习日语。

这一个句子里就包含了多个高频词：

  • 今日：时间背景。
  • 私：说话者自己。
  • は：话题。
  • 学校：地点。
  • で：动作发生场所。
  • 日本語：学习对象。
  • を：宾语标记。
  • 勉強します：学习，礼貌形。
