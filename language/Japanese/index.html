文字系统 ​

日语使用三种文字：汉字、平假名、片假名。

汉字（Kanji） ​

  • 约 2000 个汉字就占了实际书写中使用的 95% 以上。
  • 汉字对于区分句子中的不同单词至关重要。
  • 名词以及形容词和动词的词干几乎都用汉字来书写，副词也常常用汉字表示。

平假名（Hiragana） ​

平假名主要用于表达语法意义。

片假名（Katakana） ​

  • 片假名所代表的发音与平假名相同，但主要用于表示从西方国家引进的新词。
  • 也可像斜体字一样，用来突出某些词语。
  • 由于日语中没有空格，有时会使用 ・ 来表示空格。

ロック・アンド・ロール

发音（Intonation） ​

长元音 ​

可根据不同元音的特点，分别加上 あ、い 或 う 来延长相应元音的发音时间。发出长元音时，这两个音是合二为一的。

平假名：少数情况下，/e/ 元音音节通过添加 え 来延长，/o/ 元音音节通过添加 お 来延长。

片假名：所有长元音统一用一个短横 ー 来表示。

小写元音「ァ、ィ、ゥ、ェ、ォ」 ​

由于平假名音素的局限性，人们创造了这五个元音的小写形式来表示原本不存在于日语中的音素。

vwfchdtjsh
ヴァワファチャダタジャシャa
ヴィウィフィチディティジシi
ヴウフチュドゥトゥジュシュu
ヴェウェフェチェデテジェシェe
ヴォウォフォチョドトジョショo

汉字的读法 ​

几乎每个汉字都有两种不同的读法：

读法说明
音読（音读み）汉字原本的汉语读音，多用于复合词或固定搭配中
訓読（訓读み）日语中的读音，多用于单独使用的汉字
  • 某些汉字（尤其是最常见的那些）可能有多种不同的音读或训读方式。
  • 还有一些复合词的读音与各个组成汉字的读音无关，需要单独记忆。
  • 实际上，大多数单词通常只用一个汉字来表示，而且大多数汉字的读音也都不超过两种。

送假名（送り仮名） ​

訓读也用于形容词和动词中。这类词通常会附带一串假名，称为送假名。这样即使单词变成不同形态时，汉字的读音也能保持不变。

音变 ​

在复合词中，汉字的读音会稍作变化，以便于发音。常见的变化包括：

  • h 音变成 b 或 p 音
  • つ 变成 っ

重复符号「々」 ​

々 并不是真正的汉字，它只表示前面的字符需要重复书写。例如，「时时」、「様様」都可以写作「时々」、「様々」。

基础语法 ​

状态表达：だ ​

日语中没有像英语中 be 这样的表示状态的动词。在名词或な形容词后加上 だ 来表示某物的状态。

形式结构例句
肯定名词/ナ形 + だ学生だ。（是学生。）
否定名词/ナ形 + じゃない学生じゃない。（不是学生。）
过去名词/ナ形 + だった学生だった。（曾是学生。）
过去否定名词/ナ形 + じゃなかった学生じゃなかった。（曾不是学生。）

注意：

  • だ 能让句子听起来更加强调和有力，更符合陈述句的特点。
  • 男性经常在句子末尾使用 だ。
  • 完全可以不用 だ 来表达「某人是学生」这样的意思。
  • 如果从上下文就能明白主语是谁，可以不必明确指出主语。

否定形式的变形逻辑：

否定：名词/ナ形 + じゃない

过去否定：将 じゃない 的 い 去掉，加上 かった，得到 じゃなかった。

助词（Particles） ​

助词是由一个或多个平假名组成的字符，加在单词的末尾，用来确定该单词在句子中的语法功能。

主题助词「は」 ​

  • は 读作 /ha/，但作为主题助词使用时，读作 /wa/。
  • 主题助词用于表明句子所谈论的内容，即句子的 主题。

ジョンは明日。 // 至于约翰，是明天。

包含性主题助词「も」 ​

も 在话题标记的基础上增加了「也」的含义，用于引入与当前话题相关的另一个话题。

アリス：トムは学生？ ボブ：うん、トムも学生。 // 是的，汤姆也是学生。

アリス：でもトムは学生じゃない。 // 但是汤姆不是学生。 ボブ：ううん、トムも学生じゃない。 // 不，汤姆也不是学生。

指代助词「が」 ​

当我们不知道话题到底是什么时，が 就派上用场了。它可以看作在回答一个隐含的问题。

句子含义
誰が学生？谁是那个学生？（谁 = 未知信息）
学生は誰？那个学生是谁？（学生 = 已知话题）

私は学生。 // 至于我，我是学生。 私が学生。 // 我（就是）那个学生。→ 回答「谁是学生？」这个问题

「は」与「が」的区分：

助词作用语言思维
は标记主题「至于……；关于……」
が标记指代「那个……；这件事……」

形容词（Adjectives） ​

形容词可以直接修饰其后面的名词。所有形容词分为两类：ナ形容詞 和 イ形容詞。

ナ形容詞（na-adjective） ​

  • ナ形容詞本质上相当于名词。
  • 直接修饰名词时，在形容词与名词之间加 な。
  • 变形规则与名词相同。
用法例句
谓语友達は親切。 // 朋友很亲切。
修饰友達は親切な人だ。 // 朋友是个亲切的人。
否定友達は親切じゃない。 // 朋友不亲切。

「好き」的特殊用法：

ボブは魚が好きだ。 // 鲍勃喜欢鱼。

可以将 好き 理解为「令人向往的 / 想要的」。这句话的主题是「鲍勃」，而「鱼」则具体指明了鲍勃所喜欢的东西——这是主题词与指示词协同使用的典型例子。

从句修饰：

魚が好きな人。 // 喜欢鱼的人。

魚が好き、魚が好きじゃない 等整个短语都是用来修饰 人 的。

イ形容詞（i-adjective） ​

  • 所有的イ形容词都以平假名 い 结尾。
  • 直接修饰名词时无需加上 な。
  • 不能像名词否定形式那样加上陈述用的 だ。
用法例句
修饰高いレストラン // 贵的餐厅
从句修饰値段が高いレストラン // 价格贵的餐厅

例外：「いい」

所有变形都要以 よい 为基础，而不是 いい。

いい → よくない（否定） いい → よかった（过去）

以 い 结尾的ナ形容词很少，而且通常不使用汉字书写。几乎所有其他以 い 结尾的ナ形容词都用汉字书写。

动词基础（Verb Basics） ​

在日语中，动词总是位于句子的末尾。一个语法完整的句子只需要一个动词即可。

常见动词 ​

动词类型含义
いるru-动词存在（有生命物体）
あるu-动词存在（无生命物体）
着る（きる）ru-动词穿
する例外做 / 进行
くる例外来

动词分类 ​

除了两个例外动词外，所有动词都属于 ru-动词 或 u-动词 两类：

类型特征
ru-动词以 る 结尾
u-动词可以以包括 る 在内的多种 u-元音发音结尾

判断规则：

  • 不以 る 结尾 → 一定是 u-动词
  • 以 る 结尾，且 る 前面的元音是 /a/、/u/ 或 /o/ → u-动词
  • 以 る 结尾，且前面的元音是 /i/ 或 /e/ → 大多数情况下是 ru-动词

以「い」段假名结尾的 ru-动词 又称之为 上一段动词。 以「え」段假名结尾的 ru-动词 又称之为 下一段动词。

否定形 ​

将动词变为否定形式，表示「某项动作并没有发生」。

  • 对于 ru-动词：去掉「る」，加上「ない」

    食べる + ない = 食べない

  • 对于以「う」结尾的 u-动词：把「う」变成「わ」，再加上「ない」

    買う + わ + ない = 買わない

  • 对于其他所有 u-动词：把う段假名改为对应的あ段假名，再加上「ない」

    待つ → 待た + ない = 待たない

  • 特殊动词：

    1. する → しない
    2. くる → こない
    3. ある → ない

过去式 ​

ru-动词 ​

去 る + た。

u-动词 ​

按照词尾分为四类：

词尾变化例句
すす → した話す → 話した
くく → いた書く → 書いた
ぐぐ → いだ泳ぐ → 泳いだ
む/ぶ/ぬ→ んだ読む → 読んだ、遊ぶ → 遊んだ、死ぬ → 死んだ
る/う/つ→ った作る → 作った、買う → 買った、待つ → 待った

⚠ 例外： 行く → 行った（不是 行いた）

过去否定式 ​

规则对所有动词统一：取动词的否定形式 → 去掉 ない 结尾中的 い → 用 かった 替换。

行く → 行かない → 行かなかった 食べる → 食べない → 食べなかった する → しない → しなかった

与动词搭配的助词 ​

宾语助词「を」 ​

  • 在单词末尾加上 を，表示该词是动词的 直接宾语。
  • を 的发音是 /wo/，但在实际说话中听起来更接近 /o/。
  • を 这个字符从来不会用在其他地方（其片假名形式 ヲ 也几乎从不使用）。

⾼速道路を⾛る。// Run through expressway.

当名词与 する 连用时，を 可以省略，[noun + する] 结构可视为一个动词。

毎⽇、⽇本语を勉强する。// Study Japanese everyday.

目标助词「に」 ​

に 表示动词的 作用对象——动词是朝向与 に 相连的词语来发挥作用的。

对比をに
作用动词对直接宾语施加动作动词朝向目标发挥作用
搭配及物动词所有表示移动的动词
  • 所有表示移动的动词，其作用对象都由 に 来指定。
  • 目标助词始终表示「向……去」。

家に帰らない。// Not go back home. ボブは⽇本に⾏った。 // Bob went to Japan.

如果想表达「从……来」，必须使用 から。から 还经常与 まで 连用，表示「直到……为止」：

宿題を今日から明日までする。 // 作业从今天做到明天。

在日语中，「​目标」 这一概念非常通用，并不限于表示运动的动词。例如，对于表示「存在」的动词（ある和いる）​，物体的位置可以被视为其目标。时间也是一种常见的「​目标」。

「に」与存在动词：

いい友达に会った。 // Met good friend. 无生命用 ある 猫は部屋にいる。 // Cat is in room. 有生命用 いる

「に」与时间：

使用 に 后，日期变成具体的目标时间点，强调动作在那个时间发生；不使用则没有特别强调意味。

来年、日本に行く。 // 明年去日本。——一般陈述 来年に日本に行く。 // 在明年去日本。——强调时间点

方向助词「へ」 ​

  • 虽然 へ 的通常发音是 /he/，但作为助词使用时，发音始终为 /e/（え）。
助词含义
に以某个目标为最终目的地（明确）
へ正朝着该目标的方向前进（模糊）
  • へ 只能与表示方向性的动词搭配使用。
  • へ 并不能确保该目标就是最终的终点。
  • 对于没有物理移动方向的动词，不能使用 へ。
  • へ 也可以表示朝着抽象概念的方向：

勝ちへ向かう。 // 朝着胜利前进。

场景助词「で」 ​

で 帮助明确动作发生的 背景或环境。可以理解为「通过……的方式」。

映画馆で⾒た。// Saw at movie theater. レストランで昼ご饭を⾷べた。// Ate lunch at restaurant.

「何で」的读法：

何 通常读作 なに，但有时读作 なん。建议先一直使用 なに 直到有人指出何时应读 なん。

何できた？ // 怎么来的？——で 在此表示方式

なんで 是独立的词，意为「为什么」，与助词 で 无直接关系。

主题化：当地点 / 宾语成为主题时 ​

地点是主题 ​

将主题助词 は 和 も 加在表示地点的三个助词 に、へ、で 之后。

ボブ：どこで食べる？ アリス：イタリアレストランではどう？ // Bob: Where should we eat? Alice: How about Italian restaurant?

宾语是主题 ​

话题可以不用 を 标记就作为直接宾语。如果加上了 を 标记反而会变成错误表达。

❌ 日本語をは、習う。 // 错误！

及物动词与不及物动词 ​

也称为 他动词 ，自动词。

类型特征
及物动词有施事者实施的动作
不及物动词无需施事者就能发生的动作

一般而言，有：

  • 「す」结尾的动词，几乎都是他动词，其自动词往往变成「れる／える」段。

  • 「ある」结尾的动词，几乎都是自动词，其他动词往往是「える」段。

  • 「おす」 结尾的他动词，自动词多是「いる」段。

如果动词以 「える」 结尾，有时是他动词（閉める），有时是自动词（消える），需要结合对应词来判断。

及物不及物
電気を消す。 // 关灯。電気が消える。 // 灯灭了。
ボールを落とした。 // 球被弄掉了。ボールが落ちた。 // 球掉了。

关键点：

  • 不及物动词不能有直接宾语。
  • 只有表示运动的动词的地点才可以用 を 搭配不及物动词。

定语从句与语序 ​

将动词和状态词视为形容词来处理 ​

动词的否定形、过去式及过去否定式可以像形容词一样直接用来修饰名词：

形式修饰名词
动词否定食べない人 // 不吃的人
动词过去食べた人 // 吃了的人
动词过去否定食べなかった人 // 没吃的人

例外： 对于用 だ 表示的单纯非过去时态，无法如此使用（需要用 の 或 な）。

状态从句用作形容词 ​

友達じゃなかったアリスは、いい友達になった。 // 曾经不是朋友的 Alice，变成了好朋友。

动词从句修饰名词 ​

动词从句也可以像形容词一样用来修饰名词：

晩ご飯を食べなかった人は、映画で見た銀行に行った。 // 没吃晚饭的人去了在电影里看到的银行。

名词修饰名词 ​

当多个名词并非相互修饰的关系时，可以将它们直接连在一起：

国際教育センター // 国际教育中心

这些名词之间没有任何语法上的连接关系。有时候某些组合使用得如此频繁，以至于它们几乎变成了独立的单词。

日语句子语序 ​

日语句子结构的核心规则：

  • 一个完整的句子必须有谓语动词，且该动词必须放在句尾。
  • 在动词之前出现的成分可以没有固定的顺序，只要有了动词，句子就成立了。
  • 这就是使用助词的意义所在——无论单词在句子中的位置如何，助词都能明确其语法功能。
  • 除了 だ 之外，可以用定语从句来修饰名词，从而构成复合句。

私は公园でお弁当を⾷べた。// 我在公园里吃了便当。 公园で私はお弁当を⾷べた。 お弁当を私は公园で⾷べた。

名词相关助词 ​

并列助词「と」 ​

把两个或多个名词组合起来，表示「和」的意思。

本と雑志と叶书を买った。// Bought book, magazine, and post card.

と 的另一个类似用法是表示与某人或某物一起完成的动作。

友达と话した。Talked with friend.

模糊列举「や」和「とか」 ​

助词特点
や模糊列举，暗示可能还有其他未列出的事物；理解为「和/或等等」
とか与 や 意思相同，但用法更口语化

饮み物やカップやナプキンは、いらない？// You don't need (things like) drink,cup, or napkin, etc.?

靴とかシャツを买う。// Buy (things like) shoes and shirt, etc...

「の」助词 ​

の 用途繁多，是非常重要的助词。

所有格 / 名词化 ​

ボブの本。// Book of Bob.

如果上下文能够明确表明省略了什么内容，被修饰的名词就可以省略。

そのシャツは誰のシャツ？ → そのシャツは誰の？ // 那件衬衫是谁的？

の 可以直接当做名词，表示一般类事物，此时该名词并非句子重点。

⽩いのは、かわいい。// Thing that is white is cute.

⽩い物は、かわいい。Thing that is white is cute.

使用ナ形容词来修饰 の 时，仍然需要加上“な”。

静かなのが、アリスの部屋だ。// Quiet one is room of Alice.

词汇来源意思
このこれの 演变这个（近称）
そのそれ + の 缩写形式那个（中称）
あのあれの 演变那个（远称）

「の」的解释性用法 ​

在句子最后一个分句末尾加上 の，能让句子带有 解释的性质。这类句子包含着对某事原因的解释性含义，相当于「其实……」「事情是这样的……」。

形式例句
基本今は忙しいのだ。 // 其实，现在有点忙。
缩约今は忙しいんだ。 // 同上，更口语化。
疑问忙しいの？ // 你很忙吗？——男女皆可
  • 虽然 のだ 是标准形式，但通常会被 んだ 替代（因为更易于发音）。
  • 陈述句中，成年男性几乎总会加上表示肯定的 だ；疑问句中的 の 没有性别含义。

在表达状态时，当使用 の 来表示这种说明性含义时，需要加上 な, 以便将其与表示“所属关系”的含义区分开来。

ジムのだ。// It is of Jim.

ジムなのだ。 // It is Jim (with explanatory tone)

关键区别：

  • 使用说明性的 の：告诉听者「看，原因就在这里」
  • 不使用 の：只是单纯传递信息

副词与句末助词 ​

副词的构成 ​

来源规则例句
い形容词用 く 替换 い早い → 早く // 早早地
な形容词加上助词 に静か → 静かに // 安静地

注意： 并非所有副词都源自形容词。像 全然 和 たくさん 这样的词本身就是副词，无需变形即可直接使用。

场所副词示例：

図書館の中では、静かにする。 // 在图书馆里要安静。

句末助词 ​

句末助词是指总是出现在句子末尾、用来改变句子「语气」或「感觉」的助词。

「ね」——寻求认同 ​

在句子末尾加上 ね，表示寻求对方对自己所说内容的认同或赞同，相当于「对吧？」。

そうね。 // 是啊。/ 对吧。

男性可能会说 そうだね。

「よ」——传达新信息 ​

当 よ 被加在句子末尾时，表示说话者正在向听者传达一些 新信息，相当于「你知道吗……」。

ボブ：うん。でも、明⽇⾬が降るよ。// Bob: Yeah. But it will rain tomorrow, you know.

「よね」——传达新信息 + 寻求认同 ​

将 よ 和 ね 组合形成 よね，用于向听者传达自己想要表达的新观点，同时寻求对方的认同。

ボブは、⿂が好きなんだよね。// Bob, You know, you like fish.
