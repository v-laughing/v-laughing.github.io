1. 把数组中的 0 移到末尾 ​

283. Move Zeroes (Easy)

Leetcode / 力扣

html
For example, given nums = [0, 1, 0, 3, 12], after calling your function, nums should be [1, 3, 12, 0, 0].

将非0元素从头排到尾，剩下的置为0。

i表示下一个非零元素的索引值。

python
def move_zeroes(nums):
    i = 0
    for a in nums:
        if a:
            nums[i] = a
            i += 1 
    for j in range(i, len(nums)):
        nums[j] = 0

3. 洗牌算法 ​

leetcode

给定一个数组，随机的打乱它。

暴力法 拷贝该数组，每次随机选取一个数，作为当前目标数组的值，然后在拷贝数组中删除该数，继续如此。

可以证明，此方法对于每个元素，被取到的概率是一致的。

Fisher-Yates 暴力法的基础上，原地打乱。

指针i为当前索引，放置从剩下的元素中随机取出来的值。

当指针为i时，剩余的元素为[i:n]，从中随机取出一个，与a[i]交换即可。即，

a[i]=swap(a[i],a[randint(i,n−1)])
python
def shuffle(a):
	n = len(a)
	for i in range(n):
		j = random.randint(i, n-1)
		a[i], a[j] = a[j], a[i]
	return a

2. 改变矩阵维度 ​

566. Reshape the Matrix (Easy)

Leetcode / 力扣

html
Input:
nums =
[[1,2],
 [3,4]]
r = 1, c = 4

Output:
[[1,2,3,4]]

Explanation:
The row-traversing of nums is [1,2,3,4]. The new reshaped matrix is a 1 * 4 matrix, fill it row by row by using the previous list.

KEY: 第i个元素在矩阵[r, c]中的位置为（i/c，i%c）

python
def matrix_reshape(nums, r, c):
    if len(nums) * len(nums[0]) != r * c:
        return nums
    res = [[0 for i in range(c)] for j in range(r)]
    i = 0
    for row in nums:
        for a in row:
            res[i//c][i%c] = a
            i += 1
    return res

3. 找出数组中最长的连续 1 ​

485. Max Consecutive Ones (Easy)

Leetcode / 力扣

遍历计数，保存最大值

python
def findMaxConsecutiveOnes(nums):
    target = 0
    cnt = 0
    for a in nums:
        if a:
            cnt += 1
        else:
            target = max(target, cnt)
            cnt = 0
    target = max(target, cnt)
    return target

4. 有序矩阵查找 ​

240. Search a 2D Matrix II (Medium)

Leetcode / 力扣

html
[
   [ 1,  5,  9],
   [10, 11, 13],
   [12, 13, 15]
]

从右上角开始遍历

java
public boolean searchMatrix(int[][] matrix, int target) {
    if (matrix == null || matrix.length == 0 || matrix[0].length == 0) return false;
    int m = matrix.length, n = matrix[0].length;
    int row = 0, col = n - 1;
    while (row < m && col >= 0) {
        if (target == matrix[row][col]) return true;
        else if (target < matrix[row][col]) col--;
        else row++;
    }
    return false;
}

5. 有序矩阵的 Kth Element ​

378. Kth Smallest Element in a Sorted Matrix ((Medium))

Leetcode / 力扣

html
matrix = [
  [ 1,  5,  9],
  [10, 11, 13],
  [12, 13, 15]
],
k = 8,

return 13.

解题参考：Share my thoughts and Clean Java Code

二分查找解法：

java
public int kthSmallest(int[][] matrix, int k) {
    int m = matrix.length, n = matrix[0].length;
    int lo = matrix[0][0], hi = matrix[m - 1][n - 1];
    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;
        int cnt = 0;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n && matrix[i][j] <= mid; j++) {
                cnt++;
            }
        }
        if (cnt < k) lo = mid + 1;
        else hi = mid - 1;
    }
    return lo;
}

堆解法：

java
public int kthSmallest(int[][] matrix, int k) {
    int m = matrix.length, n = matrix[0].length;
    PriorityQueue<Tuple> pq = new PriorityQueue<Tuple>();
    for(int j = 0; j < n; j++) pq.offer(new Tuple(0, j, matrix[0][j]));
    for(int i = 0; i < k - 1; i++) { // 小根堆，去掉 k - 1 个堆顶元素，此时堆顶元素就是第 k 的数
        Tuple t = pq.poll();
        if(t.x == m - 1) continue;
        pq.offer(new Tuple(t.x + 1, t.y, matrix[t.x + 1][t.y]));
    }
    return pq.poll().val;
}

class Tuple implements Comparable<Tuple> {
    int x, y, val;
    public Tuple(int x, int y, int val) {
        this.x = x; this.y = y; this.val = val;
    }

    @Override
    public int compareTo(Tuple that) {
        return this.val - that.val;
    }
}

6. 一个数组元素在 [1, n] 之间，其中一个数被替换为另一个数，找出重复的数和丢失的数 ​

645. Set Mismatch (Easy)

Leetcode / 力扣

html
Input: nums = [1,2,2,4]
Output: [2,3]
html
Input: nums = [1,2,2,4]
Output: [2,3]

最直接的方法是先对数组进行排序，这种方法时间复杂度为 O(NlogN)。本题可以以 O(N) 的时间复杂度、O(1) 空间复杂度来求解。

标记法：遍历数组，将s[a]位置的元素取反。则反转两次的位置i即是重复的值，没有反转的位置j即是丢失的值。两个位置上的值都为正。

python
def findErrorNums(nums):
    nums.insert(0, 0)
    res = []
    for i, a in enumerate(nums):
        a = abs(a)
        if nums[a] < 0:
            res.append(a)
        nums[a] = -nums[a]    
    for i, a in enumerate(nums):
        if a > 0 and i != res[0]:
            res.append(i)
    return res

7. 找出数组中重复的数，数组值在 [1, n] 之间 ​

287. Find the Duplicate Number (Medium)

Leetcode / 力扣

要求不能修改数组，也不能使用额外的空间。

二分查找解法：

java
public int findDuplicate(int[] nums) {
     int l = 1, h = nums.length - 1;
     while (l <= h) {
         int mid = l + (h - l) / 2;
         int cnt = 0;
         for (int i = 0; i < nums.length; i++) {
             if (nums[i] <= mid) cnt++;
         }
         if (cnt > mid) h = mid - 1;
         else l = mid + 1;
     }
     return l;
}

双指针解法: 类似于有环链表中找出环的入口。 将所给数组构造成一个链表，应用如下规则：下标为i的节点的下一个节点为nums[i]。由此，问题转化为求有环链路中的环起始点节点。

python
def findDuplicate(nums):
    slow = nums[0]
    fast = nums[nums[0]]
    while slow != fast:
        slow = nums[slow]
        fast = nums[nums[fast]]
    slow = 0
    while slow != fast:
        slow = nums[slow]
        fast = nums[fast]
    return slow

8. 数组相邻差值的个数 ​

667. Beautiful Arrangement II (Medium)

Leetcode / 力扣

html
Input: n = 3, k = 2
Output: [1, 3, 2]
Explanation: The [1, 3, 2] has three different positive integers ranging from 1 to 3, and the [2, 1] has exactly 2 distinct integers: 1 and 2.

题目描述：数组元素为 1~n 的整数，要求构建数组，使得相邻元素的差值不相同的个数为 k。

显然，符合的排列方式不止一种。 驼峰排列法： 以1, n, 2, n-1... 方式构成k-1种差值，然后按递增（k为奇数）或递减（k为偶数）排列剩下的整数，这构成1种差值。

python
def constructArray( n, k) :
    res = []
    m = k//2
    for i in range(1, m + 1):
        res.append(i)
        res.append(n-i+1)
    rest = range(n-m, m, -1) if k%2==0 else range(m+1, n-m+1, 1)   
    for i in rest:
        res.append(i)
    return res

9. 数组的度 ​

697. Degree of an Array (Easy)

Leetcode / 力扣

html
Input: [1,2,2,3,1,4,2]
Output: 6

题目描述：数组的度定义为元素出现的最高频率，例如上面的数组度为 3。要求找到一个最小的子数组，这个子数组的度和原数组一样。

要考虑多个相同最高频率情况。

java
public int findShortestSubArray(int[] nums) {
    Map<Integer, Integer> numsCnt = new HashMap<>();
    Map<Integer, Integer> numsLastIndex = new HashMap<>();
    Map<Integer, Integer> numsFirstIndex = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int num = nums[i];
        numsCnt.put(num, numsCnt.getOrDefault(num, 0) + 1);
        numsLastIndex.put(num, i);
        if (!numsFirstIndex.containsKey(num)) {
            numsFirstIndex.put(num, i);
        }
    }
    int maxCnt = 0;
    for (int num : nums) {
        maxCnt = Math.max(maxCnt, numsCnt.get(num));
    }
    int ret = nums.length;
    for (int i = 0; i < nums.length; i++) {
        int num = nums[i];
        int cnt = numsCnt.get(num);
        if (cnt != maxCnt) continue;
        ret = Math.min(ret, numsLastIndex.get(num) - numsFirstIndex.get(num) + 1);
    }
    return ret;
}

10. 对角元素相等的矩阵 ​

766. Toeplitz Matrix (Easy)

Leetcode / 力扣

html
1234
5123
9512

In the above grid, the diagonals are "[9]", "[5, 5]", "[1, 1, 1]", "[2, 2, 2]", "[3, 3]", "[4]", and in each diagonal all elements are the same, so the answer is True.

题目描述：判断是否为Toeplitz Matrix。

按正常顺序来遍历数组，对于每个遍历到的数字，都跟其右下方的数字对比，如果不相同，直接返回false即可。可不遍历最后一行和最后一列。

python
def isToeplitzMatrix(matrix):
    for i in range(len(matrix) -1 ):
        for j in range(len(matrix[0]) -1 ):
            if matrix[i][j] != matrix[i+1][j+1]:
                return False
    return True

11. 嵌套数组 ​

565. Array Nesting (Medium)

Leetcode / 力扣

html
Input: A = [5,4,0,3,1,6,2]
Output: 4
Explanation:
A[0] = 5, A[1] = 4, A[2] = 0, A[3] = 3, A[4] = 1, A[5] = 6, A[6] = 2.

One of the longest S[K]:
S[0] = {A[0], A[5], A[6], A[2]} = {5, 6, 2, 0}

题目描述：S[i] 表示一个集合，集合的第一个元素是 A[i]，第二个元素是 A[A[i]]，如此嵌套下去。求最大的 S[i]。

数组以节点i，以{i, A[i]}为边构成一个多环图。即求得此环图的最大深度。

KEY: 对同一环上的节点而言，其深度遍历所得的深度是一样的

python
def arrayNesting(nums):
    target = 0
    for i in range(len(nums)):
        cnt = 0
        j = i
        while nums[j] != -1:
            t = nums[j]
            nums[j] = -1
            j = t
            cnt += 1
        target = max(target, cnt)
    return target

12. 分隔数组 ​

769. Max Chunks To Make Sorted (Medium)

Leetcode / 力扣

html
Input: arr = [1,0,2,3,4]
Output: 4
Explanation:
We can split into two chunks, such as [1, 0], [2, 3, 4].
However, splitting into [1, 0], [2], [3], [4] is the highest number of chunks possible.

题目描述：分隔数组，使得对每部分排序后数组就为有序。

显然，在i点分割数组时，应该满足：

i=max{arr[j]},0≤j≤i
python
def maxChunksToSorted(arr):
    cnt = 0
    max_a = 0
    for i, a in enumerate(arr):
        max_a = max(max_a, a)
        if i == max_a:
            cnt += 1
    return cnt
