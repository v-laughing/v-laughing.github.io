    4. 两个字符串包含的字符是否完全相同 ​

    242. Valid Anagram (Easy)

    Leetcode / 力扣

    html
    s = "anagram", t = "nagaram", return true.
s = "rat", t = "car", return false.

    可以用 HashMap 来映射字符与出现次数，然后比较两个字符串出现的字符数量是否相同。

    由于本题的字符串只包含 26 个小写字符，因此可以使用长度为 26 的整型数组对字符串出现的字符进行统计，不再使用 HashMap。

    使用字典

    python
    def isAnagram(s, t):
	smap = collections.Counter(s)
	tmap = collections.Counter(t)
	return smap == tmap

    5. 计算一组字符集合可以组成的回文字符串的最大长度 ​

    409. Longest Palindrome (Easy)

    Leetcode / 力扣

    html
    Input : "abccccdd"
Output : 7
Explanation : One longest palindrome that can be built is "dccaccd", whose length is 7.

    使用长度为 256 的整型数组来统计每个字符出现的个数，每个字符有偶数个可以用来构成回文字符串。

    因为回文字符串最中间的那个字符可以单独出现，所以如果有单独的字符就把它放到最中间。

    python
    def longestPalindrome(s):
	smap = collections.Counter(s)
	count = reduce(lambda x, y: (x//2)*2 + (y//2)*2, smap.values())
	if count < len(s):
		count += 1
	return count

    6. 字符串同构 ​

    205. Isomorphic Strings (Easy)

    Leetcode / 力扣

    html
    Given "egg", "add", return true.
Given "foo", "bar", return false.
Given "paper", "title", return true.

    将两字符串的当前字符映射到所在位置，再次遇到这对字符是其映射值应该相等，如果是同构的话。

    python
    def isIsomorphic(s, t):
	smap = {}
	tmap = {}
	i = 0
	for a, b in zip(s, t):
		if smap.get(a) != tmap.get(b):
			return False
		smap[a] = i  
		tmap[b] = i
		i += 1  
	return True

    7. 最长回文子字符串 ​

    5. Longest Palindromic Substring(Medium)

    力扣

    html
    Input: "babad"
Output: 'aba'

    动态规划 令dp[i][j]在区间[i, j]的字符串是否为回文。则有，

    dp[i][j]=(s[i]==s[j])&dp[i+1][j−1],i<j

    记录求解过程中的最大值即可。

    python
    def longestPalindrome(s):
	n = len(s)
	dp = [[False]*n for i in range(n)]
	wi = 0; wj = 0;
	for j in range(n):
		for i in range(j+1):
			if s[j] != s[i]:
				continue
			dp[i][j] = True if j - i < 3 else dp[i+1][j-1]
			if dp[i][j] and (j - i > wj - wi) :
				wi, wj = i, j
	return s[wi:wj+1]

    中心扩展 思路同下一题。

    python
    def countSubstrings(String s) {
	n = len(s)
	wi, wj = 0, 0
    def extendSubstrings(start, end):
		nonlocal wi, wj
	    while start >= 0 and end < n and s[start] == s[end]:
        	start -= 1
        	end += 1
		if end - start - 2 > wj - wi:
			wi, wj = start + 1, end - 1
	for i in range(n):
        extendSubstrings(i, i)     # 奇数长度
        extendSubstrings(i, i + 1) # 偶数长度
    return s[wi:wj+1]

    8. 回文子字符串个数 ​

    647. Palindromic Substrings (Medium)

    Leetcode / 力扣

    html
    Input: "aaa"
Output: 6
Explanation: Six palindromic strings: "a", "a", "a", "aa", "aa", "aaa".

    从字符串的某一位开始，尝试着去扩展子字符串。

    python
    def countSubstrings(String s) {
	n = len(s)
	cnt = 0
    def extendSubstrings(start, end):
		nonlocal cnt
	    while start >= 0 and end < n and s[start] == s[end]:
        	start -= 1
        	end += 1
        	cnt += 1
		
	for i in range(n):
        extendSubstrings(i, i)     # 奇数长度
        extendSubstrings(i, i + 1) # 偶数长度
    return cnt

    9. 判断一个整数是否是回文数 ​

    9. Palindrome Number (Easy)

    Leetcode / 力扣

    要求不能使用额外空间，也就不能将整数转换为字符串进行判断。

    将整数分成左右两部分，右边那部分需要转置，然后判断这两部分是否相等。

    需要考虑到数字长度的奇偶情况。

    java
    public boolean isPalindrome(int x) {
    if (x == 0) {
        return true;
    }
    if (x < 0 || x % 10 == 0) {
        return false;
    }
    int right = 0;
    while (x > right) {
        right = right * 10 + x % 10;
        x /= 10;
    }
    return x == right || x == right / 10;
}

    10. 统计二进制字符串中连续 1 和连续 0 数量相同的子字符串个数 ​

    696. Count Binary Substrings (Easy)

    Leetcode / 力扣

    html
    Input: "00110011"
Output: 6
Explanation: There are 6 substrings that have equal number of consecutive 1's and 0's: "0011", "01", "1100", "10", "0011", and "01".

    一个这样1{n}0{m}或0{n}1{m}的字符串，包含满足要求的子字符串的个数为min{m,n}。因此，问题转换为求所有这些长字符串长度。这些长字符串首尾相交。

    html
    0011
  1100
    0011
显然，共计6个
    python
    def countBinarySubstrings(s):
    preLen = 0; curLen = 1; count = 0; 
	for i in range(1, len(s)):
		if s[i] != s[i-1]:
			preLen = curLen
			curLen = 1
		else:
			curLen += 1
		if preLen >= curLen:
			count += 1
    return count
