    1. 分配饼干 ​

    455. Assign Cookies (Easy)

    Leetcode / 力扣

    html
    Input: grid[1,3], size[1,2,4]
Output: 2

    题目描述：每个孩子都有一个满足度 grid，每个饼干都有一个大小 size，只有饼干的大小大于等于一个孩子的满足度，该孩子才会获得满足。求解最多可以获得满足的孩子数量。

    贪心策略：

    1. 给一个孩子的饼干应当尽量小并且又能满足该孩子，这样大饼干才能拿来给满足度比较大的孩子。
    2. 因为满足度最小的孩子最容易得到满足，所以先满足满足度最小的孩子。

    在以上的解法中，我们只在每次分配时饼干时选择一种看起来是当前最优的分配方法，但无法保证这种局部最优的分配方法最后能得到全局最优解。我们假设能得到全局最优解，并使用反证法进行证明，即假设存在一种比我们使用的贪心策略更优的最优策略。如果不存在这种最优策略，表示贪心策略就是最优策略，得到的解也就是全局最优解。

    证明：假设在某次选择中，贪心策略选择给当前满足度最小的孩子分配第 m 个饼干，第 m 个饼干为可以满足该孩子的最小饼干。假设存在一种最优策略，可以给该孩子分配第 n 个饼干，并且 m < n。我们可以发现，经过这一轮分配，贪心策略分配后剩下的饼干一定有一个比最优策略来得大。因此在后续的分配中，贪心策略一定能满足更多的孩子。也就是说不存在比贪心策略更优的策略，即贪心策略就是最优策略。


    python
    def find_content_children(grid, size):
    grid.sort()
    size.sort()
    gn = len(grid)
    gi = 0;
    for s in size:
        if gi >= gn:
            break
        if s >= grid[gi]:
            gi += 1
    return gi

    2. 不重叠的区间个数 ​

    435. Non-overlapping Intervals (Medium)

    Leetcode / 力扣

    html
    Input: [ [1,2], [1,2], [1,2] ]

Output: 2

Explanation: You need to remove two [1,2] to make the rest of intervals non-overlapping.
    html
    Input: [ [1,2], [2,3] ]

Output: 0

Explanation: You don't need to remove any of the intervals since they're already non-overlapping.

    题目描述：计算让一组区间不重叠所需要移除的区间个数。

    转化为贪心问题：找到尽可能多的区间，使其构成不重叠的区间。 然后用区间总个数减去不重叠区间的个数。

    贪心策略：使得每次选择的区间的结尾部分值尽可能小。

    按区间的结尾进行排序，每次选择结尾最小，并且和前一个区间不重叠的区间。

    python
    def erase_overlap_intervals(intervals):
    if not intervals:
        return 0
    intervals.sort(key=lambda inte: inte[1])
    end = intervals[0][0] - 1
    cnt = 0 
    for s, e in intervals:
        if s >= end:
            cnt += 1
            end = e 
    return len(intervals) - cnt

    3. 投飞镖刺破气球 ​

    452. Minimum Number of Arrows to Burst Balloons (Medium)

    Leetcode / 力扣

    Input:
[[10,16], [2,8], [1,6], [7,12]]

Output:
2

    题目描述：气球在一个水平数轴上摆放，可以重叠，飞镖垂直投向坐标轴，使得路径上的气球都被刺破。求解最小的投飞镖次数使所有气球都被刺破。

    也是计算不重叠的区间个数，不过和 Non-overlapping Intervals 的区别在于，[1, 2] 和 [2, 3] 在本题中算是重叠区间。

    python
    def find_min_arrow_shots(points):
    if not points:
        return 0
    points.sort(key=lambda inte: inte[1])
    end = points[0][0] - 1
    cnt = 0 
    for s, e in points:
        if s > end:
            cnt += 1
            end = e 
    return cnt

    4. 根据身高和序号重组队列 ​

    406. Queue Reconstruction by Height(Medium)

    Leetcode / 力扣

    html
    Input:
[[7,0], [4,4], [7,1], [5,0], [6,1], [5,2]]

Output:
[[5,0], [7,0], [5,2], [6,1], [4,4], [7,1]]

    题目描述：一个学生用两个分量 (h, k) 描述，h 表示身高，k 表示排在前面的有 k 个学生的身高比他高或者和他一样高，排序原数组使其满足这种排序方式。

    为了使插入操作不影响后续的操作，身高较高的学生应该先做插入操作，否则身高较小的学生原先正确插入的第 k 个位置可能会变成第 k+1 个位置。

    身高 h 降序、个数 k 值升序，然后将某个学生插入队列的第 k 个位置中。

    python
    def reconstruct_queue(people):
    people.sort(key=lambda p: (-p[0], p[1]))
    res = []
    for p in people:
        res.insert(p[1], p)
    return res

    5. 买卖股票最大的收益 ​

    121. Best Time to Buy and Sell Stock (Easy)

    Leetcode / 力扣

    题目描述：一次股票交易包含买入和卖出，只进行一次交易，求最大收益。

    令p[i]表示第i天卖出可获得的最大收益，a[i]为第i天股票价格，则有：

    p[i]=a[i]−min{a[i]},0≤i≤n−1target=max{p[i]},0≤i≤n−1
    python
    def max_profit(prices):
    if not prices:
        return 0
    cur_min = prices[0]
    target = 0
    for a in prices:
        if a < cur_min:
            cur_min = a
        target = max(a-cur_min, target)
    return target

    6. 买卖股票的最大收益 II ​

    122. Best Time to Buy and Sell Stock II (Easy)

    Leetcode / 力扣

    题目描述：可以进行多次交易，多次交易之间不能交叉进行，可以进行多次交易。

    贪心策略： 只要此次交易有收益，就交易。

    考虑三个点，且 a <= b <= c，有两种情况：

    1. 在b，c点交易，则总收益为 (c - b) + (b - a) 。
    2. 仅在c点交易，则总收益为c - a。

    两种情况是等同的。

    python
    def max_profit2(prices):
    target = 0
    cur_hold = prices[0]
    for a in prices:
        if a > cur_hold:
            target += a - cur_hold
        cur_hold = a
    return target

    7. 种植花朵 ​

    605. Can Place Flowers (Easy)

    Leetcode / 力扣

    html
    Input: flowerbed = [1,0,0,0,1], n = 1
Output: True

    题目描述：flowerbed 数组中 1 表示已经种下了花朵。花朵之间至少需要一个单位的间隔，求解是否能种下 n 朵花。

    KEY: 变量will_place表i−1位置是否为0。

    NOTICE: 要考虑后一个数

    python
    def can_place_flowers(flowerbed, n):
    will_place = True
    l = len(flowerbed)
    for i in range(l):
        if n <= 0:
            break
        if flowerbed[i]:
            will_place = False
        else:
            if will_place:
                if i+1 >= l or not flowerbed[i+1]:
                    n -= 1
            will_place = not will_place 
    return n == 0

    8. 判断是否为子序列 ​

    392. Is Subsequence (Medium)

    Leetcode / 力扣

    html
    s = "abc", t = "ahbgdc"
Return true.

    双指针解法

    python
    def is_subsequence(s, t):
    j = 0
    s_len = len(s)
    if s_len == 0:
        return True
    for a in t:
        if a == s[j]:
            if j == s_len - 1:
                return True
            j += 1
    return False

    9. 修改一个数成为非递减数组 ​

    665. Non-decreasing Array (Easy)

    Leetcode / 力扣

    html
    Input: [4,2,3]
Output: True
Explanation: You could modify the first 4 to 1 to get a non-decreasing array.

    题目描述：判断一个数组是否能只修改一个数就成为非递减数组

    KEY: 遍历索引为i时，保证0~i内数组是有序的。 when nums[i] < nums[i - 1] ：

    1. 令nums[i - 1] = nums[i]，为了不影响了后续操作，nums[i]应尽可能小。
    2. 当 nums[i] < nums[i - 2]，修改 nums[i - 1] = nums[i] 不能使数组成为非递减数组，只能修改 nums[i] = nums[i - 1]。
    python
    def check_possibility(nums):
    cnt = 0
    for i in range(1, len(nums)):
        if cnt >= 2:
            break
        if nums[i] >= nums[i-1]:
            continue
        else:
            cnt += 1
            if i >= 2 and nums[i] < nums[i-2]:
                nums[i] = nums[i-1]
            else:
                nums[i-1] = nums[i]
    return cnt <= 1

    10. 子数组最大的和 ​

    53. Maximum Subarray (Easy)

    Leetcode / 力扣

    html
    For example, given the array [-2,1,-3,4,-1,2,1,-5,4],
the contiguous subarray [4,-1,2,1] has the largest sum = 6.

    解析： 令c[i]表示以i位置结尾的最大子数组的和，a[i]表数组i位置的值，则有

    c[i]=max{c[i−1]+a[i],a[i]},1≤i≤n−1target=max{c[i]|0≤i≤n−1}
    python
    def max_subarray(nums):
    cur_sum = nums[0]
    target = cur_sum
    for i in range(1, len(nums)):
        cur_sum = max(cur_sum+nums[i], nums[i])
        target = max(cur_sum, target)
    return target

    11. 分隔字符串使同种字符出现在一起 ​

    763. Partition Labels (Medium)

    Leetcode / 力扣

    html
    Input: S = "ababcbacadefegdehijhklij"
Output: [9,7,8]
Explanation:
The partition is "ababcbaca", "defegde", "hijhklij".
This is a partition so that each letter appears in at most one part.
A partition like "ababcbacadefegde", "hijhklij" is incorrect, because it splits S into less parts.

    题目描述：如题

    当遇到元素a时，此段分割出的字符串的尾边界最近是最右边的元素a所在的位置。

    记L[a]表a元素在字符串的最右边的索引，则Sij是一个子串的条件为：

    j=max{L[S[k]]},i≤k≤j
    python
    def partition_labels(S):
    Lindex = {a : i for i, a in enumerate(S)}
    res = []
    i = max_lindex = 0
    for j, a in enumerate(S):
        max_lindex = max(max_lindex, Lindex[a])
        if max_lindex == j:
            res.append(j - i + 1)
            i = j + 1
    return res
